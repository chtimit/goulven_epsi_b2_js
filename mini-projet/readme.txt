Ce dossier contient le mini-projet(sujet 2)

Pour l'exécuter :
ouvrir un bash dans le dossier mini-projet(là où est ce readme)
exécuter la commande : 
	npm install
(cela va installé les extention nécéssaire)(on pourait les installer une à une mais c'est long et chiant)
puis :
	node app.js
(cela va lancer les services)
si tout va bien, windows vous demandra d'autoriser l'application, répondez oui.
sur un navigateur compatible, tapez "localhost:3000" dans la bare d'adresse
suivez les instructions à l'écrant et ça marche en parti
